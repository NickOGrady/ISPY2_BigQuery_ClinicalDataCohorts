# ISPY2_BigQuery_ClinicalDataCohorts

This is an RShiny application that generates a table of select clinical data. User selects variables to view, interested cohort (e.g. HER+/-, HR+/-), and desired drug arm(s). Use of application highlighted in "PRoBE the Cloud Toolkit: Finding the Best Biomarkers of Drug Response within a Breast Cancer Clinical Trial."

User will need a Google Cloud Platform project, and relevant data uploaded to BigQuery. Application authorizes user persmissions through email.  User will need to enter specific project variables inside << >> notation.
